SISTEMA ATM EM C



Projeto desenvolvido para a disciplina Data Structures and Algorithm, usando a linguagem C.



O sistema simula um caixa eletrônico simples pelo terminal, permitindo consultar saldo, realizar saque, depósito, ver extrato e encerrar o sistema.

SOBRE O PROJETO



O projeto usa conceitos básicos de C, como funções, switch-case, laços de repetição, estruturas de decisão e modularização.



O código foi separado em arquivos para ficar mais organizado.
